# Scrum Board – Frontend

Detta är frontend-delen av Scrum board slutprojekt.

## 🚀 Funktioner
- Skapa tasks
- Flytta tasks mellan New / Ongoing / Done
- Ta bort tasks
- Tilldela medlem till task
- Hämtar data från backend API

## 🛠️ Tekniker
- React
- Vite
- Fetch API
- CSS

## ▶️ Starta projektet

```bash
npm install
npm run dev
